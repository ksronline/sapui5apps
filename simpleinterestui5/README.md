# Simple Interest Calculator - SAP UI5

A SAP UI5 application that calculates simple interest based on principal amount, rate of interest, and time period.

## Formula

Simple Interest = (Principal x Rate x Time) / 100

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. The application will open in your default browser.

## Features

- Input fields for Principal Amount, Rate of Interest (%), and Time Period (years)
- Calculate button to compute the simple interest
- Clear button to reset all fields
- Displays calculated Simple Interest and Total Amount (Principal + Interest)
